## React Component for rendering text

A basic div that formats two paragraph elements and an svg if passed one.

### Component usage

```js
<MessageCard title="Message Title">
  Descriptor text adding extra information and context
</MessageCard>
```

### Using props to customize the text

Modify the text to see it change live:

```js live
<MessageCard icon={<Icon />} title="Message Title">
  Descriptor text adding extra information and <br /> context
</MessageCard>
```
