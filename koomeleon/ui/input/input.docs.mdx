## React Component for rendering text

An input component that takes a prefix, suffix, and an invalid state with an error message.

This is still a regular React HTML input underneath meaning any attribute that may be added
in react or in HTML may be added here also.

### Component usage

```js
const BasicInput = () => {
  const [value, setValue] = useState('');

  return <Input onChange={(e) => setValue(e.target.value)} value={value} />;
};
```
