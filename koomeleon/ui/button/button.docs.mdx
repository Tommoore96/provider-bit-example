## React Component for rendering text

A basic button with text

### Component usage

```js
<Button>Click me</Button>
```

### Using props to customize the text

Modify the text to see it change live:

```js live
() => {
  const buttonStyles = {
    margin: '10px',
  };

  return (
    <div style={{ alignItems: 'center', display: 'flex' }}>
        <Button style={buttonStyles} buttonSize="small" buttonType="primary">
          Small
        </Button>
        <Button style={buttonStyles} buttonSize="small" buttonType="primary">
          <ClockIcon />
          Small
        </Button>
        <Button style={buttonStyles} buttonSize="medium" buttonType="primary">
          Medium
        </Button>
        <Button style={buttonStyles} buttonSize="medium" buttonType="primary">
          <ClockIcon />
          Medium
        </Button>
    </div>
  );
};
```
