## React Component for rendering error messages

A basic div that renders an error message and an Icon.

The icon is defined in the theme file on the `assets.icons.error` property.

### Component usage

```js
<ErrorMessage>Extreme error!</ErrorMessage>
```

### Using props to customize the text

Modify the text to see it change live:

```js live
  <ErrorMessage>Extreme error!</ErrorMessage>
```
